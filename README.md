# Box Direct Link Copy

[English README](README_en.md)

Box のリスト表示・グリッド表示で、共有リンクではない「直リンク」をその場でコピーできる Chrome 拡張です。

- フォルダ: `https://<tenant>.app.box.com/folder/{id}`
- ファイル: `https://<tenant>.app.box.com/file/{id}`

## できること

- リスト表示で、ファイル名/フォルダ名の横に「直リンクをコピー」アイコンを表示
- グリッド表示で、カード左上に「直リンクをコピー」アイコンを表示
- 行/カードの「その他のオプション」メニューに `直リンクをコピー` を追加
- コピー結果をアイコン付近のトーストで表示
- UIメッセージをロケールで自動切替（`ja`: 日本語 / その他: 英語）

## 使い方

### リスト表示

- ファイル名/フォルダ名横の鎖アイコンをクリック
- または `...` メニュー内の `直リンクをコピー` をクリック

### グリッド表示

- カード左上の鎖アイコンをクリック
- または `...` メニュー内の `直リンクをコピー` をクリック

## インストール（開発者モード）

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパー モード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このリポジトリのディレクトリを選択
5. Box の一覧画面を開いて動作確認

## 既知の制約

- 対象は Box Web (`https://app.box.com/*`, `https://*.app.box.com/*`) の一覧UIのみ
- Box 側の DOM / UI 変更で動作が変わる場合があります
- コピー先URLは「アクセス権があるユーザー向けの直リンク」です（権限自体は拡張機能で変更しません）
